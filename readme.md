# not-welcome-page

A captive portal page shown to short-term rental guests (e.g. Airbnb) connecting to a residential WiFi network.

## What it does

Displays a message explaining why short-term rentals harm residential neighborhoods: rent pressure, displacement, loss of community. Not directed at individual guests, but at the structural problem.

## How it works

The WiFi SSID is named "AirBnB Free WIFI" to attract guests. Before internet access is granted, this page is shown as a captive portal. No tracking, no forms, no JavaScript.

## Content

- Available in English (default) and German via a pure CSS language switch
- Argues from a resident's perspective, not the host's
